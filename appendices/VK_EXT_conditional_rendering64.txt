=== Description

This extension allows the execution of one or more rendering commands to be
conditional on a 64-bit value in buffer memory, contrary to
VK_EXT_conditional_rendering which only allows 32-bit values.

=== Issues

1) Should VK_EXT_conditional_rendering64 require VK_EXT_conditional_rendering?

*RESOLVED*: No, some hardware might just support 64-bit predicates.
